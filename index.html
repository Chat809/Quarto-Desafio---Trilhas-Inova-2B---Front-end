Perguntas Frequentes

Os dados são atualizados?

Sim, utilizamos APIs oficiais como IBGE, Brasil.IO e IPEA, garantindo dados atualizados e confiáveis.

Preciso pagar para acessar?

Não. A plataforma é gratuita, aberta e de acesso público.

Posso utilizar os dados nos meus projetos?

Sim! Desde que você cite as fontes dos dados, é permitido utilizar para pesquisas, reportagens e projetos.

Quais indicadores estão disponíveis?

Renda média, escolaridade, homicídios, saneamento básico, população e densidade demográfica.
